# Taller · Simuladores CompanyGame

Presentación interactiva (HTML) para **docentes** sobre los simuladores de negocio de
**CompanyGame** como recurso pedagógico para la toma de decisiones en Ciencias Económicas.

Distribuidor: **Simuladores de Negocios Colombia** (Grupo Edutec).

## Contenido

1. Portada
2. Cartelera de simuladores
3. Ingreso a la plataforma
4. ¿Qué es CompanyGame? (el recurso)
5. El ciclo de aprendizaje experiencial
6. Los módulos del simulador
7. Competencias que desarrolla
8. Estrategia pedagógica (ciclo de Kolb, debriefing…)
9. CompanyGame en el mundo
10. El RETO — Competencia Internacional
11. El Torneo ACBSP·CompanyGame
12. Métodos de implementación
13. Preguntas frecuentes
14. Cierre y contacto

## Cómo verla

Abre `index.html` en cualquier navegador, o visita la versión publicada en GitHub Pages
(ver el enlace en la sección **About/Acerca de** del repositorio).

## Tecnología

HTML5 + CSS3 + JavaScript vanilla, todo embebido en `index.html`. Sin dependencias.
Navegación por flechas, teclado y swipe. Imágenes en `assets/`.
